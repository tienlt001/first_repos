# Tic-Tac-Toe (CLI)

Play Tic-Tac-Toe from the command line.

Usage:

- Run:

  python -m tic_tac_toe.cli

- Choose mode: two players or vs AI, follow prompts.

Run the web UI (requires Flask):

- Install dependencies: `pip install -r requirements.txt`
- Start the server: `python -m tic_tac_toe.web` or `FLASK_APP=tic_tac_toe.web:app flask run`
- Open http://127.0.0.1:5000 in your browser and play.
